# vue-cli 를 통한 프로젝트 생성
1. vue-cli 설치
```
npm install -g @vue/cli
// g 는 global 설치 옵션
```
2. vue create 프로젝트이름
```
vue create front-end
```
3. vue-cli 옵션화면에서 필요한 옵션을 선택
